# PRODIGY_DS_03

## Task 03 - Decision Tree Classifier

This project is part of my Data Science Internship at **ProDigy InfoTech**.

## Objective

Build a Decision Tree Classifier to predict outcomes using demographic or behavioral data.

## Dataset Used

Breast Cancer Wisconsin Dataset (Scikit-learn)

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Project Workflow

- Loaded the dataset
- Explored dataset information
- Checked for missing values
- Split data into training and testing sets
- Trained a Decision Tree Classifier
- Evaluated model accuracy
- Generated Classification Report
- Created Confusion Matrix
- Visualized Feature Importance
- Plotted the Decision Tree

## Results

- Model Accuracy: **96%**
- Successfully classified the dataset using Decision Tree.
